## Installation

The Python package `poochrepositories` can be installed from PyPI:

```
python -m pip install poochrepositories
```

## Development installation

If you want to contribute to the development of `poochrepositories`, we recommend
the following editable installation from this repository:

```
python -m pip install --editable .[tests]
```

Having done so, the test suite can be run using `pytest`:

```
python -m pytest
```
